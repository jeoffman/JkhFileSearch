Most of the time the built-in Windows file search is slow and doesn't find the files I am looking for.

I used to use the free version of Agent Ransack but there were a couple of annoying "features" so I decided to write my own.

It has Windows Explorer Right-Click context, so you can right click a folder in Explorer and bring up a file search:
